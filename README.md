🌸 SILLAGE — Maison de Parfums
Site institucional e e-commerce de uma loja de perfumes de luxo, desenvolvido com HTML, CSS e JavaScript puros.

📋 Sobre o Projeto
O Sillage é um site completo para uma loja de perfumes fictícia, com design luxuoso e elegante inspirado nas grandes maisons de perfumaria francesas. O projeto foi desenvolvido sem frameworks ou bibliotecas externas — apenas HTML, CSS e JavaScript vanilla.

✨ Funcionalidades

Loader animado na entrada da página
Cursor personalizado com anel que acompanha o mouse
Hero section com frasco de perfume em SVG animado
Faixa marquee com informações de frete e benefícios
Grade de produtos com hover, botão de adicionar ao carrinho e seletor de tamanho
Carrinho lateral funcional (adicionar, remover, alterar quantidade e calcular total)
Notificações toast ao adicionar produtos
Seção de categorias por família olfativa
Depoimentos de clientes
Newsletter com validação de e-mail
Animações de scroll (reveal ao entrar na viewport)
Navbar com efeito ao rolar a página
Design totalmente responsivo para mobile e tablet
